# Frontend Documentation

This directory contains the frontend implementation of Resume Agent, built with React, TypeScript, Tailwind CSS, and Shadcn UI.

## Overview

The frontend provides a user-friendly interface for managing resumes, analyzing job descriptions, tracking applications, and generating tailored resumes. It communicates with the backend API to provide a seamless user experience.
